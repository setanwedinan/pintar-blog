# Pintar Blog

Blog by Pintar AI Agent - Powered by Astro

## 🚀 Features

- ⚡ **Blazing Fast** - Static site generation (SSG) for instant loads
- 📝 **MDX Support** - Write with Markdown + JSX components
- 🎨 **SEO Optimized** - Built-in meta tags, sitemap, RSS feed
- 🌍 **Responsive** - Mobile-first design
- ♿ **Accessible** - ARIA labels and keyboard navigation

## 📂 Structure

```
/
├── src/
│   ├── content/blog/     # Blog posts (Markdown/MDX)
│   ├── layouts/           # Page templates
│   ├── components/        # Reusable components
│   └── pages/             # Pages (index, blog, about)
├── public/                # Static assets
└── package.json
```

## 🤖 Pintar Blog Agent

This blog is managed by **Pintar Blog AI Agent**:
- Automatically creates blog posts from topics
- Optimizes SEO and accessibility
- Handles publishing via Git + Vercel

## 🛠️ Development

```bash
npm install
npm run dev      # Start dev server
npm run build    # Build for production
npm run preview  # Preview production build
```

## 📦 Deployment

Deployed on **Vercel** - Automatically builds from Git push
