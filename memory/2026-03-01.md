### Key Learnings

1. **Hybrid Approach**: HuggingFace API + keyword fallback = robust solution
   - API: High accuracy (~94%) but unreliable (rate-limited)
   - Fallback: Good accuracy (86%) and always available
   - System auto-switches based on availability

2. **Context Mapping Critical**: Generic sentiment needs domain-specific mapping
   - POSITIVE about USD ≠ POSITIVE about IDR
   - FX sentiment: USD up = IDR down (inverse relationship)

3. **Confidence Scores Valuable**: Not just label, but how confident
   - Dashboard now shows 0-1 confidence for each news item
   - Helps users weigh information importance

4. **Testing Essential**: Verify both success and fallback paths
   - Model accuracy ≠ production accuracy
   - Timeout handling prevents cascade failures
   - Edge cases reveal limitations (multi-entity sentences)

### Technical Details

**HuggingFace Inference API**:
- Free tier, no API key needed
- Rate-limited (caused timeouts in testing)
- 10s timeout with auto-fallback

**Keyword Classifier**:
- 14 positive keywords, 14 negative keywords
- Multi-keyword detection with 60-90% confidence scoring
- Indonesian + English support

**Context Detection**:
- Simple keyword matching: USD/Dollar vs IDR/Rupiah
- Works well for headlines, struggles with multi-entity sentences
