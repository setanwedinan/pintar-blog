# 2026-02-19 - Knowledge RAG Project

## Knowledge RAG - Personal Knowledge Base (Phase 1 MVP)

**Project**: Personal RAG knowledge base with semantic search, entity extraction, and URL ingestion

**Faizal's Request** (from Telegram topic):
- Build personal knowledge base with RAG
- Ingest URLs from Telegram topic (https://t.me/c/3743758645/37)
- Support: Articles, YouTube videos, Twitter threads, PDFs
- Extract entities (people, companies, concepts)
- SQLite storage with vector embeddings
- Natural language queries with semantic search
- Time-aware ranking (recent = boost)
- Source-weighted ranking
- Chrome Relay for paywalled sites

**Phase 1 MVP Implementation** (completed today):
- Web article ingestion via `web_fetch`
- Entity extraction (LLM-based, placeholder)
- Vector embeddings via OpenAI API (text-embedding-3-small, 1536 dimensions)
- JSON-based storage with lowdb (avoided SQLite due to native dependencies)
- Cosine similarity search
- CLI interface for testing

**Tech Stack Decisions**:
1. **Storage**: lowdb (JSON file-based)
   - Initially tried SQLite + sqlite-vector
   - sqlite-vector package not found in npm registry
   - Tried sqlite-vec - requires better-sqlite3 (native compilation)
   - better-sqlite3 failed: needs `make` tool (not available in sandbox)
   - **Decision**: Use lowdb (pure JS, no native deps)

2. **Embeddings**: OpenAI API
   - Model: text-embedding-3-small
   - Dimensions: 1536
   - Cost: ~$0.02 per 1M tokens

3. **Search**: Cosine similarity algorithm
   - Hand-coded implementation
   - Works with Float32Array embeddings

**NVIDIA API Integration**:
- Model: nvidia/nv-embedqa-e5-v5
- Asymmetric embeddings (query vs passage)
- 768-dimensional vectors (smaller than OpenAI's 1536)
- Free tier via NVIDIA NIM API
- API endpoint: https://integrate.api.nvidia.com/v1

**Configuration**:
